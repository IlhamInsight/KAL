# Folder Gambar untuk Halaman Eigenfaces

Tempatkan screenshot/grafik hasil visualisasi program Anda di folder ini dengan nama-nama berikut agar otomatis terdeteksi oleh file markdown:

1. `hsil.png` - Hasil output hitungan SVD di konsol/terminal.
2. `hasil10eigenfaces.png` - Grafik matplotlib visualisasi 10 Eigenfaces pertama.
3. `hasil_scatterplot.png` - Grafik scree plot retensi energi singular.

Pastikan format file adalah `.png` dan penulisan nama file sesuai dengan daftar di atas.
